# 🎬 Movie Recommendation System

## 📌 About the Project

This project is a movie recommendation system built using machine learning.
It helps users find movies based on genre, rating, year, and director.

---

## 🚀 Features

* 🎯 Movie Recommendation (Content-Based)
* ⭐ Rating-Based Recommendation (SVD + KNN)
* 📊 Data Analysis Dashboard (Power BI)
* 💰 Revenue Prediction (Random Forest)

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn
* Plotly
* Power BI

---

## 📂 Dataset

* IMDb Dataset (`cleaned_imdb.csv`)

---

## ▶️ How to Run

pip install -r requirements.txt
streamlit run app.py
